## 문제 18
### **다음 중 각 문서를 임베딩(벡터화)하지 않고 키워드 빈도 기반으로 유사도를 측정하는 방식은?**

| 보기 | 설명 |
|------|------|
| 1. Word2Vec | |
| **2. BM25** | |
| 3. Cosine Similarity | |
| 4. BERT Embedding | |
| 5. OpenAI Embedding | |

**정답: 2.**

**해설:**
1. **오답** - 단어를 고정 차원의 밀집 벡터(dense vector)로 임베딩하는 방식입니다. 벡터화를 수행하므로 정답이 아닙니다.
2. **정답** - 정답입니다. 문서를 벡터로 임베딩하지 않고 단어 빈도(Term Frequency)와 역문서 빈도(IDF)를 기반으로 문서-쿼리 간 관련성 점수를 직접 계산하는 렉시컬 검색 방식입니다. 대표적인 키워드 기반 검색 알고리즘입니다.
3. **오답** - 두 벡터 간의 코사인 각도를 이용해 유사도를 측정하는 방법입니다. 벡터화된 문서에 적용하는 유사도 측정 방식이므로, 임베딩이 선행되어야 합니다.
4. **오답** - BERT 트랜스포머 모델을 사용하여 문맥을 고려한 밀집 벡터로 임베딩하는 시맨틱 검색 방식입니다.
5. **오답** - OpenAI API(text-embedding-ada-002 등)를 통해 텍스트를 밀집 벡터로 변환하는 시맨틱 임베딩 방식입니다.
