## 문제 1
### **다음 중 텍스트, 이미지, 음성 등 다양한 형식의 데이터를 동시에 처리할 수 있는 모델 유형은?**

| 보기 | 설명 |
|------|------|
| ① LLM (Large Language Model) | 대규모 언어 모델, 텍스트 처리 특화 |
| **② LMM (Large Multimodal Model)** | 텍스트·이미지·음성 등 복합 데이터 처리 |
| ③ RNN (Recurrent Neural Network) | 순환 신경망, 순차 데이터에 특화 |
| ④ CNN (Convolutional Neural Network) | 합성곱 신경망, 이미지 처리에 주로 사용 |

**정답: ②**

**해설:**
LMM(Large Multimodal Model)은 텍스트뿐 아니라 이미지, 오디오, 비디오 등 다양한 유형의 데이터를 함께 처리할 수 있는 AI 모델입니다.

- **LLM**: 텍스트만 처리
- **LMM**: 여러 형식(멀티모달) 처리

## 문제 2
### **Pandas 라이브러리에서 2차원 테이블 형태의 자료구조는?**

| 보기 | 설명 |
|------|------|
| ① Series | 1차원 인덱스+값 구조 |
| **② DataFrame** | 행과 열로 이루어진 2차원 테이블 |
| ③ Array | NumPy의 다차원 배열 |
| ④ Tensor | PyTorch/TensorFlow의 자료구조 |

**정답: ②**

**해설:**
Pandas의 핵심 자료구조:

| 자료구조 | 차원 | 설명 |
|----------|------|------|
| **Series** | 1차원 | 인덱스 + 값 배열 |
| **DataFrame** | 2차원 | 행/열 테이블, 엑셀과 유사 |

## 문제 3
### **머신러닝에서 모델이 훈련 데이터에 지나치게 맞춰져 새로운 데이터에 대한 성능이 낮아지는 현상은?**

| 보기 | 설명 |
|------|------|
| ① 언더피팅 (Underfitting) | 모델이 훈련 데이터도 제대로 학습하지 못한 상태 |
| **② 과적합 (Overfitting)** | 훈련 데이터에 과도하게 맞춰진 상태 |
| ③ 정규화 (Normalization) | 데이터 스케일을 조정하는 전처리 기법 |
| ④ 드롭아웃 (Dropout) | 과적합 방지를 위한 정규화 기법 |

**정답: ②**

**해설:**
과적합(Overfitting)은 모델이 훈련 데이터의 **노이즈까지 학습**하여 일반화 능력이 떨어지는 현상입니다.

```
훈련 정확도 ↑↑  vs  테스트 정확도 ↓↓  →  과적합
```

방지 방법: 드롭아웃, L1/L2 정규화, 데이터 증강, 조기 종료
