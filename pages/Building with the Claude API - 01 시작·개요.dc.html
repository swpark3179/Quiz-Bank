Claude 모델 개요

Overview of Claude models

Claude는 서로 다른 우선순위에 최적화된 세 가지 모델 패밀리를 제공한다. 모든 모델은 핵심 능력(텍스트 생성·코딩·이미지 분석)을 공유하며, 차이는 최적화 초점에 있다.

Opus
최고 지능
깊은 추론·계획이 필요한 복잡한 다단계 작업에 적합. 비용·지연(latency)이 높음.
Sonnet
균형
지능·속도·비용의 균형. 강력한 코딩과 정밀한 코드 편집. 대부분의 실무 사례에 최적.
Haiku
속도·비용
가장 빠르고 저렴. Opus/Sonnet 같은 추론 능력은 없음. 실시간 상호작용·대량 처리에 적합.
